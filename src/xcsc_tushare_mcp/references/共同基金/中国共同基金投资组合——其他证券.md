## 中国共同基金投资组合——其他证券

___

接口：other\_portfolio

描述：记录基金定期（季度）公布的持有其他证券明细

限量：单次最大10000条

**输入参数**

|名称|类型|必选|描述|
|---|---|---|---|
|ts\_code|str|N|基金代码|
|end\_date|str|N|截止日期(yyyymmdd)|

**输出参数**

|名称|类型|默认显示|描述|
|---|---|---|---|
|ts\_code|str|Y|基金代码|
|end\_dt|str|Y|截止日期|
|ann\_dt|str|Y|公告日期|
|s\_info\_holdwindcode|str|Y|关联WindCustomCode可获取证券基本信息|
|value|float|Y|持仓市值|
|quantity|float|N|持仓数量（股/张）|
|valuetonav|float|N|持仓市值占基金净值比例(%)|

**接口示例**

```python

df=pro.other_portfolio(ts_code='005021.OF',end_date='20181231')
```

**数据示例**

```

1  005021.OF  20181231  ...          IC1903.CFE  1635600
```
