## 指数成分和权重

___

接口：index\_weight

描述：获取各类指数成分和权重，月度数据 ，建议输入参数里开始日期和结束日分别输入当月第一天和最后一天的日期。

限量：单次最大10000条

**输入参数**

|名称|类型|必选|描述|
|---|---|---|---|
|index\_code|str|N|指数代码|
|trade\_date|str|N|交易日期|
|start\_date|str|N|开始日期|
|end\_date|str|N|结束日期|

**输出参数**

|名称|类型|默认显示|描述|
|---|---|---|---|
|index\_code|str|Y|指数代码|
|con\_code|str|Y|成份代码|
|trade\_date|str|Y|交易日期|
|weight|float|Y|权重|

**接口示例**

```python
pro = ts.pro_api(your token)
df = pro.index_weight(index_code='399300.SZ', start_date='20180901', end_date='20180930')
```
