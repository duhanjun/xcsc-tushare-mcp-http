## 港股通每月成交统计

___

接口：ggt\_monthly

描述：港股通每月成交信息，测试环境数据从201910开始

限量：单次最大10000条

**输入参数**

|名称|类型|必选|描述|
|---|---|---|---|
|month|str|N|月度（格式YYYYMM，下同，支持多个输入）|
|start\_month|str|N|开始月度|
|end\_month|str|N|结束月度|
|market\_type|int|N|市场类型 1：沪股通 2：港股通（沪） 3：深股通 4：港股通（深）|

**输出参数**

|名称|类型|默认显示|描述|
|---|---|---|---|
|month|str|Y|统计月度|
|markettype|int|Y|市场类型 1：沪股通 2：港股通（沪） 3：深股通 4：港股通（深）|
|day\_buy\_amt|float|Y|当月日均买入成交金额（亿元），沪股通、深股通币种人民币，港股通币种港元|
|day\_buy\_vol|float|Y|当月日均买入成交笔数（万笔）|
|day\_sell\_amt|float|Y|当月日均卖出成交金额（亿元），沪股通、深股通币种人民币，港股通币种港元|
|day\_sell\_vol|float|Y|当月日均卖出成交笔数（万笔）|
|total\_buy\_amt|float|Y|总买入成交金额（亿元），沪股通、深股通币种人民币，港股通币种港元|
|total\_buy\_vol|float|Y|总买入成交笔数（万笔）|
|total\_sell\_amt|float|Y|总卖出成交金额（亿元），沪股通、深股通币种人民币，港股通币种港元|
|total\_sell\_vol|float|Y|总卖出成交笔数（万笔）|

**接口示例**

```python


#获取单月全部统计
df = pro.ggt_monthly(month='202006')

#获取多月统计信息
df = pro.ggt_monthly(month='202006,202007,202009')

#获取时间段统计信息
df = pro.ggt_monthly(start_month='202001', end_month='202005')
```
