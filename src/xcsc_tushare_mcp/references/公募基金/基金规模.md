## 公募基金规模

___

接口：fund\_share

描述：获取基金规模数据，包含上海和深圳ETF基金

限量：单次最大3000条

**输入参数**

|名称|类型|必选|描述|
|---|---|---|---|
|ts\_code|str|N|基金代码，支持多只基金同时提取，用逗号分隔|
|trade\_date|str|N|交易日期（YYYYMMDD）|
|start\_date|str|N|开始日期|
|end\_date|str|N|结束日期|
|fund\_type|str|N|基金类型，见下表|
|market|str|N|市场：SH/SZ|

**输出参数**

|名称|类型|默认显示|描述|
|---|---|---|---|
|ts\_code|str|Y|基金代码|
|trade\_date|str|Y|交易日期|
|fd\_share|float|Y|基金份额（万）|
|fund\_type|str|Y|基金类型|
|market|str|Y|市场|

**fund\_type标识说明**

|标识|含义|
|---|---|
|ETF|ETF基金|
|LOF|LOF基金|
|SF|分级基金|
|CEF|封闭基金|

**代码示例**

```python

#单只基金
df = pro.fund_share(ts_code='150018.SZ')

#多只基金
df = pro.fund_share(ts_code='150018.SZ,150008.SZ')
```
