## 可转债有条件回售价格和触发比例

___

接口：bond\_repur\_pr

描述：获取可转债有条件回售的价格和触发比例信息

限量：单次最大10000条

**输入参数**

|名称|类型|必选|描述|
|---|---|---|---|
|ts\_code|str|N|转债代码|
|trade\_date|str|N|交易日期（yyyymmdd）|
|start\_date|str|N|开始日期（yyyymmdd）|
|end\_date|str|N|结束日期（yyyymmdd）|

**输出参数**

|名称|类型|默认显示|描述|
|---|---|---|---|
|ts\_code|str|Y|转债代码|
|repurchaseprice|float|Y|回售价|
|bgndt|str|Y|起始日期|
|enddt|str|Y|截止日期|
|trnsrt|float|Y|触发比例(%)|

**接口示例**

```python
df=pro.bond_repur_pr(trade_date='20200421')
```

**数据示例**

```

     ts_code  repurchaseprice     bgndt     enddt  trnsrt
0  128012.SZ              103  20200421  20220421      70
```
