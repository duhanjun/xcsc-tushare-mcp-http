## 中国共同基金投资组合——行业配置

___

接口：fund\_ind\_portfolio

描述：记录基金定期（季度）公布的持股分属于各行业的组合。行业分类采用证监会标准

限量：单次最大10000条

**输入参数**

|名称|类型|必选|描述|
|---|---|---|---|
|ts\_code|str|N|基金代码|
|end\_date|str|N|截止日期（yyyymmdd）|
|f\_start\_date|str|N|截止开始日期（yyyymmdd）|
|f\_end\_date|str|N|截止结束日期（yyyymmdd）|

**输出参数**

|名称|类型|默认显示|描述|
|---|---|---|---|
|ts\_code|str|Y|基金代码|
|f\_prt\_enddate|str|Y|截止日期|
|s\_info\_csrcinduscode|str|Y|证监会行业编号|
|f\_prt\_indusvalue|float|Y|持有行业市值(元)|
|f\_prt\_industonav|float|N|持有行业市值占基金净值比例(%)|
|f\_prt\_indposvalue|float|Y|积极投资持有行业市值(元)|
|s\_info\_csrcindusname|str|N|行业名称|
|opdate|datetime|N|更新时间|

**接口示例**

```python

df=pro.fund_ind_portfolio(ts_code='000001.OF',end_date='20181231')
```
