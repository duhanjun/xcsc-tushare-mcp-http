## 融资融券交易汇总

___

接口：margin

描述：获取融资融券每日交易汇总数据

限量：单次最大10000条

**输入参数**

|名称|类型|必选|描述|
|---|---|---|---|
|trade\_date|str|N|交易日期|
|exchange\_id|str|N|交易所代码（SSE上交所SZSE深交所）|
|start\_date|str|N|开始日期|
|end\_date|str|N|结束日期|

**输出参数**

|名称|类型|默认显示|描述|
|---|---|---|---|
|trade\_date|str|Y|交易日期|
|exchange\_id|str|Y|交易所代码|
|rzye|float|Y|融资余额(元)|
|rzmre|float|Y|融资买入额(元)|
|rzche|float|Y|融资偿还额(元)|
|rqye|float|Y|融券余额(元)|
|rqyl|float|Y|融券余量|
|rqmcl|float|Y|融券卖出量(股,份,手)|
|rzrqye|float|Y|融资融券余额(元)|

**接口使用**

```python

df = pro.margin(trade_date='20180802')

#或

df = pro.query('margin', trade_date='20180802', exchange_id='SSE')
```
