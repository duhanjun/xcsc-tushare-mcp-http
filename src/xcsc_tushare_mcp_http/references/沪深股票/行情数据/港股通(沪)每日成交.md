## 港股通(沪)每日成交

___

接口：ggt\_sh\_daily

描述：获取上交所发布的港股通（沪）每日成交信息，币种为人民币

限量：单次最大10000条

**输入参数**

|名称|类型|必选|描述|
|---|---|---|---|
|trade\_date|str|N|交易日期 （格式YYYYMMDD，下同。支持单日和多日输入）|
|start\_date|str|N|开始日期|
|end\_date|str|N|结束日期|

**输出参数**

|名称|类型|默认显示|描述|
|---|---|---|---|
|trade\_date|str|Y|交易日期|
|markettype|int|N|市场类型 默认为2：港股通（沪）|
|buy\_amount|float|Y|买入成交金额（亿元）|
|buy\_volume|float|Y|买入成交笔数（万笔）|
|sell\_amount|float|Y|卖出成交金额（亿元）|
|sell\_volume|float|Y|卖出成交笔数（万笔）|

**接口示例**

```python


pro = ts.pro_api()

#获取单日成交信息
df = pro.ggt_sh_daily(trade_date='20200605')

#获取多日成交信息
df = pro.ggt_sh_daily(trade_date='20200605,20200716,20200826')

#获取时间段成交信息
df = pro.ggt_sh_daily(start_date='20200601', end_date='20200701')
```
