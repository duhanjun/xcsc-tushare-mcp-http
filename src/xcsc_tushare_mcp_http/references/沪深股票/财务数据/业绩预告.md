## 业绩预告

___

接口：forecast

描述：获取业绩预告数据

限量：单次最大10000条

**输入参数**

|名称|类型|必选|描述|
|---|---|---|---|
|ts\_code|str|N|ts代码|
|ann\_date|str|N|公告日期|
|start\_date|str|N|公告开始日期|
|end\_date|str|N|公告结束日期|
|period|str|N|报告期(每个季度最后一天的日期，比如20171231表示年报)|
|type|str|N|预告类型|

**输出参数**

|名称|类型|默认显示|描述|
|---|---|---|---|
|ts\_code|str|Y|ts代码|
|ann\_date|str|Y|公告日期|
|end\_date|str|Y|报告期|
|type|float|Y|业绩预告类型|
|sign\_change|str|Y|是否变脸|
|p\_change\_min|float|Y|预告净利润变动幅度下限（%）|
|p\_change\_max|float|Y|预告净利润变动幅度上限（%）|
|net\_profit\_min|float|Y|预告净利润下限（万元）|
|net\_profit\_max|float|Y|预告净利润上限（万元）|
|last\_parent\_net|float|Y|上年同期归母净利润|
|first\_ann\_date|str|Y|首次公告日|
|summary|str|Y|业绩预告摘要|
|change\_reason|str|Y|业绩变动原因|

**接口用法**

```python

pro = ts.pro_api()
df = pro.forecast(ann_date='20190131', fields='ts_code,ann_date,end_date,type,p_change_min,p_change_max,net_profit_min')
```
