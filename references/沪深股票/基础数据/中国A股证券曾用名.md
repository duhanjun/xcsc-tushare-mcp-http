## 中国A股证券曾用名

___

接口：previous\_name

描述：记录A股名称历次更名情况，包括特别处理更名

限量：单次最大10000条

**输入参数**

|名称|类型|必选|描述|
|---|---|---|---|
|ts\_code|str|N|股票代码|
|start\_date|str|N|开始日期|
|end\_date|str|N|结束日期|
|begin\_date|str|N|起始日期|

**输出参数**

|名称|类型|默认显示|描述|
|---|---|---|---|
|ts\_code|str|Y|股票代码|
|begindate|str|Y|起始日期|
|enddate|str|Y|截至日期|
|ann\_dt|str|Y|公告日期|
|s\_info\_name|str|Y|证券简称|
|changereason|float|Y|变动原因代码 详见附录|

**接口示例**

```python

df=pro.previous_name(ts_code='600599.SH')
```

**数据示例**

```

     ts_code begindate   enddate    ann_dt s_info_name  changereason
0  600599.SH  20200611      None  20200610       *ST熊猫     200004000
1  600599.SH  20150420  20200610  20150414        熊猫金控     200001000
```
