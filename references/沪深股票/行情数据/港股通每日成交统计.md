## 港股通每日成交统计

___

接口：ggt\_daily

描述：获取港股通每日成交信息，数据从20200416开始

限量：单次最大3000条

**输入参数**

|名称|类型|必选|描述|
|---|---|---|---|
|trade\_date|str|N|交易日期 （格式YYYYMMDD，下同。支持单日和多日输入）|
|start\_date|str|N|开始日期|
|end\_date|str|N|结束日期|
|market\_type|str|N|市场类型 1：沪股通 2：港股通（沪） 3：深股通 4：港股通（深）|

**输出参数**

|名称|类型|默认显示|描述|
|---|---|---|---|
|trade\_date|str|Y|交易日期|
|markettype|int|Y|市场类型 1：沪股通 2：港股通（沪） 3：深股通 4：港股通（深）|
|buy\_amount|float|Y|买入成交金额（亿元），币种：沪股通、深股通人民币，港股通港元|
|buy\_volume|float|Y|买入成交笔数（万笔）|
|sell\_amount|float|Y|卖出成交金额（亿元），币种：沪股通、深股通人民币，港股通港元|
|sell\_volume|float|Y|卖出成交笔数（万笔）|
|fx\_buy\_price|float|Y|买入结算汇兑比率|
|fx\_sell\_price|float|Y|卖出结算汇兑比率|

**接口示例**

```python


pro = ts.pro_api()

#获取单日全部统计
df = pro.ggt_daily(trade_date='20200605')

#获取多日统计信息
df = pro.ggt_daily(trade_date='20200605,20200716,20200826')

#获取时间段统计信息
df = pro.ggt_daily(start_date='20200601', end_date='20200701')
```
