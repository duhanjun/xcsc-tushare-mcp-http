## 融资融券交易明细

___

接口：margin\_detail

描述：获取沪深两市每日融资融券明细

限量：单次最大10000条

**输入参数**

|名称|类型|必选|描述|
|---|---|---|---|
|trade\_date|str|N|交易日期|
|ts\_code|str|N|股票代码|
|start\_date|str|N|开始日期|
|end\_date|str|N|结束日期|

**输出参数**

|名称|类型|默认显示|描述|
|---|---|---|---|
|trade\_date|str|Y|交易日期|
|ts\_code|str|Y|股票代码|
|name|str|Y|股票名称|
|rzye|float|Y|融资余额（元）|
|rqye|float|Y|融券余额（元）|
|rqyl|float|Y|融券余量（手）|
|rzmre|float|Y|融资买入额（元）|
|rzche|float|Y|融资偿还额（元）|
|rqmcl|float|Y|融券卖出量（手）|
|rqchl|float|Y|融券偿还量（手）|
|rzrqye|float|Y|融资融券余额（元）|

**接口示例**

```python

df = pro.margin_detail(trade_date='20180802')

#或

df = pro.query('margin_detail', trade_date='20180802')
```
